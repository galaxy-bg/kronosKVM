# Hardware

The prototype carrier is marked `CM4-DUAL-ETH-WIFI6-BASE` and contains a
Raspberry Pi Compute Module 4.

Observed labels include HDMI0, HDMI1, USB1-3, ETH0-1, USB-C SLAVE, DISP, TF,
CAM0, CAM1, RTC battery, fan and M.2 mounting positions. Electrical functions
are provisional until inventory and operator-assisted mapping confirm them.
